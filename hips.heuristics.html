class hips.heuristics.AbstractDiving(mip_model: hips.models._mip_model.MIPModel, current_best_objective: Optional[float] = None)¶

Bases: hips.heuristics._heuristic.Heuristic

Implements an abstract version of diving heuristics. It consists of a general framework for branch-and-bound-tree traversal and stopping criteria. The branching criteria should be implemented in a corresponding subclass. This implementation roughly follows the algorithm seen in Berthold_Primal_Heuristics_For_Mixed_Integer_Programs.pdf; Page 17, “Algorithm 1: A General Diving heuristic”;

class hips.heuristics.FeasibilityPump(mip_model: hips.models._mip_model.MIPModel, T=0.5, seed=None)¶

Bases: hips.heuristics._heuristic.Heuristic

The feasibility pump is a heuristic for solving integer programs (MIPs) proposed by Fischetti et al. in 2005 ( https://link.springer.com/article/10.1007/s10107-004-0570-3). The basic idea is to repeatedly solve the MIP’s relaxation, i.e. the MIP without integer constraints, round the obtained relaxed solution to the nearest integer and then find a solution that minimizes the distance to that rounded point. The process is repeated until a feasible point to the original MIP is found or some other stopping criterion is satisfied. Further, small perturbations are used to combat cycling.

This class implements the basic feasibility pump as described in the paper above.

compute(max_iter=None, t=None, alpha=None)¶

Starts the computation of the feasibility pump.

Parameters

  • max_iter – The maximum number of iterations that should be performed. If the number of iterations is reached, the computation terminated.

  • T – Perturbation parameter.

  • alpha – If alpha is set, the feasibility pump will try to optimize towards the objective function. The higher alpha, the more we optimize towards the original objective function.

This heuristic is implemented after the scipopt primal heuristic ‘heur_bounds.h’. The scipopt library describes this as a “heuristic which fixes all integer variables to a bound (lower/upper) and solves the remaining LP “.
