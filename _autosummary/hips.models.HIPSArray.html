hips.models.HIPSArray¶

class hips.models.HIPSArray(arg)¶

Wrapper class for numpy array

This class acts as a wrapper for a numpy array. The wrapping is necessary for overriding the different Python operators. The underlying array can be accessed with the array attribute. For more details, please consult the numpy documentation.

Example
>>> import numpy as np
>>> from hips.models import HIPSArray
>>> # Creating a 2x2 HIPSArray with ones
>>> HIPSArray((2, 2))
HIPSArray([[1. 1.]
 [1. 1.]])
>>> # Creating a 3x3 identity matrix
>>> HIPSArray(np.identity(3))
HIPSArray([[1. 0. 0.]
 [0. 1. 0.]
 [0. 0. 1.]])
