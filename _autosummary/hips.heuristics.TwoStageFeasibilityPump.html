hips.heuristics.TwoStageFeasibilityPump¶

class hips.heuristics.TwoStageFeasibilityPump(model: hips.models._mip_model.MIPModel, **kwargs)¶

Implementation of feasibility pump with two stages

This class implements a version of the feasibility pump that is applicable to general mixed integer programs, i.e. programs containing binary and integer variables, as proposed by [BFL07]. In contrast, the original feasibility pump [FGL05] is only designed for mixed integer programs containing binary variables only. However, note that our implementation of the original feasibility pump is able to work with general mixed-integer programs.

The basic idea of this feasibility pump is to handle binary and integer variables in two different stages. In the first stage, the integer variables are ignored, i.e. treated as if they were continuous, and only the binary variables are considered. The first stage is completed once a feasible solution w.r.t. the binary variables has been found or when the maximum iteration has been reached. Afterwards, in the second stage, the integer variables are considered. Note that the original feasibility pump can also handle integer variables, but might be less suited for this purpose.
