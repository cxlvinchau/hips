hips.heuristics._abstract_diving.AbstractDiving¶

class hips.heuristics._abstract_diving.AbstractDiving(mip_model: hips.models._mip_model.MIPModel, current_best_objective: Optional[float] = None)¶

Implements an abstract version of diving heuristics_tmp. It consists of a general framework for branch-and-bound-tree traversal and stopping criteria. The branching criteria should be implemented in a corresponding subclass. This implementation roughly follows the algorithm seen in :cite:t:’2006:berthold’ on page 17.
