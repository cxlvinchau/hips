hips.models.LinExpr¶

class hips.models.LinExpr(vars={}, coefficients={})¶

Representation of a linear expression

This class implements a representation of a linear expression. Suppose we have variables \(x_0, \dots, x_n\) and coefficients \(a_0, \dots, a_n\) with dimensions \(d_0, \dots, d_n\). Then a linear expression of the form \({a_0}^T x_0 + \dots + {a_n}^T x_n\) can be represented with this class. For more details, please refer to hips.models.Variable.

Typically, a linear expression is not created explicitly with constructor, but rather implicitly.
