hips.solver.GurobiSolver¶

class hips.solver.GurobiSolver¶

Implementation of a linear programming solver using the Gurobi solver.

This class implements an LP solver using the Gurobi solver [GurobiOptimizationLLC21]. This solver needs a gurobipy installation and does not work without it. Please note that you also need an active Gurobi license.
