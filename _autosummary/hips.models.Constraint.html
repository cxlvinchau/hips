hips.models.Constraint¶

class hips.models.Constraint(lhs, comparator, rhs)¶

Representation a linear constraint

This class implements a representation of a linear constraint of a linear program. Typically, a constraint is not created with the constructor but rather implicitly.

__init__(lhs, comparator, rhs)¶

Constructor

Parameters

  • lhs – Left-hand side of the constraint. Has to be an instance of hips.models.LinExpr

  • comparator – A comparator, i.e. less equal, equal or greater equal (see hips.Comparator)

  • rhs – Right-hand side of the constraint. Has to be an instance of hips.models.LinExpr
