hips.models.LPModel¶

class hips.models.LPModel(lp_solver)¶

Representation of a linear program

This class implements a representation of a linear program. It provides a simple interface to create, update and optimize linear programs.

add_variable(name, lb=0, ub=None, dim=1)¶

Adds a variable to the linear program

This method adds a variable to the linear program. Variables can be high-dimensional and can have lower and higher bounds. For more details, please see hips.models.Variable.

Parameters

  • name – Name of the variable

  • lb – Lower bound of the variable. By default the lower bound is \(0\). To set the bound to \(-\infty\) set lb to None. lb has to be an instance of hips.models.HIPSArray if the variable has a dimension greater than \(1\)

  • ub – Upper bound of the variable. By default the upper bound is \(\infty\). ub has to be an instance of hips.models.HIPSArray if the variable has a dimension greater than \(1\)

  • dim – Dimension of the variable. By default the variable has dimension \(1\)

Returns

A variable, an instance of hips.models.Variable
